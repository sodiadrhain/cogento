# Changelog

All notable changes to the "cogento" extension will be documented in this file.

## [0.1.0] - 2026-02-23

### Added

- Multi-provider support (OpenAI, Anthropic, Gemini).
- Agentic tool system (File Read/Write, Command Execution, Workspace Search).
- Workspace `@file` mentions with auto-complete.
- Multimodal support (Image attachments).
- Conversation history and persistence.
- Premium, borderless Chat UI in Sidebar.
- Improved Error handling with direct settings shortcuts.
- Interactive user history (Click to restore, Copy code).
